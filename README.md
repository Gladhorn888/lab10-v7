# lab10-v7
libstack — динамический стек целых чисел

Библиотека на C реализует стек (LIFO) с автоматическим расширением буфера.
Публичный интерфейс описан в include/stack.h, скрытая реализация — в src/stack.c.
Собирается как разделяемый объект (libstack.so), может использоваться из C и Python через ctypes.

Требования

Компилятор C (GCC с поддержкой -std=c99 и -fPIC)

make

Python 3 (для тестов и вызова)

clang-tidy (опционально, для статического анализа)

doxygen (опционально, для генерации документации)

Структура проекта

.
├── include/

│ └── stack.h # публичный интерфейс с Doxygen-комментариями

├── src/

│ └── stack.c # реализация стека

├── test/

│ ├── test.c # C-тесты

│ └── test.py # Python-тесты через ctypes

├── Makefile

├── Doxyfile

└── README.md

Сборка

Для сборки динамической библиотеки выполните:

make

Будет создан файл libstack.so.

Запуск тестов

C-тесты

make test

Скомпилирует и запустит тестовую программу, проверяющую создание, push/pop/peek, обработку ошибок и расширение.

Python-тесты

make python_test

Выполнит скрипт test/test.py, который через ctypes загружает libstack.so и проверяет все основные операции.
